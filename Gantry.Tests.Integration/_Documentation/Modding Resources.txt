C# and .NET YouTube Tutorials
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

 • Tim Corey: https://www.youtube.com/user/IAmTimCorey
    - Tim teaches basic programming techniques, and methodologies.

 • Nick Chapsas: https://www.youtube.com/c/Elfocrash
    - Nick teaches intermediate to advanced C# techniques, and pattens.

 • Raw Coding: https://www.youtube.com/c/RawCoding
    - Raw Coding is a one-stop-shop for design patterns, and coding theory.

 • Feya Holmér: https://www.youtube.com/c/Acegikmo
    - Freya is one of the best teachers online, for learning how to write Shaders, and for learning Advanced Mathematics for Game Development.
